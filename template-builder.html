Resume sections

Add, remove, or reorder sections. Use Bulleted blocks for jobs, projects, education, etc. Use Labeled lines for any skill-style content with your own labels (technical, languages, certifications, tools, …). End each role line with dates (e.g. …Acme, Engineer May 2023 to Present) for right-aligned dates in the preview. AI import fills a sections array; tailoring preserves your section ids.

Extension settings · Required slots (reference)

Required data-resume-slot values

  • summary — one element (e.g. <p>)
  • Bulleted sections: each <ul> uses data-resume-slot="sec-{id}-{n}" (id starts with a letter). The preceding section title has data-resume-section-kind="bullets".
  • Labeled-line sections: value spans use data-resume-slot="skill-{id}-{n}"; the section title uses data-resume-section-kind="skills".
