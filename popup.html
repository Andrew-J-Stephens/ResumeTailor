Resume Tailor

Built-in resume · A4 PDF downloads

Layout and styling come from the extension’s built-in resume template. The AI only adjusts wording to fit the role. On a job posting, select the job description (plain text is sent—no page HTML). Tailored files download as PDF (CompanyName-resume-YYYY-MM-DD.pdf when the employer is known). If automatic PDF fails, a print preview tab opens instead.

To change layout or styling, edit src/lib/resume-template.html in the project and rebuild the extension.
